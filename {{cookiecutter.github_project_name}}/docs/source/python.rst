

Python Bindings
=================

We use pybind11_ to create the python bindings.
The python subfolder contains all the code related 
to the python bindings.

::

    project
    ├── ...
    ├── python          
    │   ├── module
    │   ├── src
    │   ├── test
    ├── ...


.. _pybind11: https://github.com/pybind/pybind11
