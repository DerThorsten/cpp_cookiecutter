Features
============
    
    
    * CMake build system
    * C++ Unit Tests with `Doctest <https://cpp-cookiecutter.readthedocs.io/en/latest/unit_tests.html>`_ 
    * `Benchmark code with google benchmark <https://cpp-cookiecutter.readthedocs.io/en/latest/benchmark.html>`_
    * Continous Integration:
        * `azure-pipelines <https://dev.azure.com/{{cookiecutter.azure_user_name}}/{{cookiecutter.github_project_name}}/_build/latest?definitionId=1&branchName=master>`_
        * `circle-ci <https://circleci.com/gh/{{cookiecutter.github_user_name}}/{{cookiecutter.github_project_name}}/tree/master>`_
        * `travis-ci <https://travis-ci.org/{{cookiecutter.github_user_name}}/{{cookiecutter.github_project_name}}>`_
    * bumpversion for version handling
    * Documentation with sphinx breathe and on
      `readthedocs <http://{{cookiecutter.readthedocs_package_name}}.readthedocs.io/en/latest/?badge=latest>`_
    * Conda Recipe Included
    * Python bindings are created via `pybind11 <https://github.com/pybind/pybind11>`_
